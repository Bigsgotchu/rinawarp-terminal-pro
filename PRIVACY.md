# RinaWarp Privacy

RinaWarp Terminal Pro is designed for developer trust. The app must not collect or transmit the private contents of your work.

## Anonymous Operational Telemetry

RinaWarp may send minimal anonymous operational telemetry to help improve reliability:

- anonymous install ID
- app version
- operating system platform
- CPU architecture
- lightweight operational counters such as task started, task completed, task failed, rollback triggered, and approval denied

This helps answer whether installs, updates, and trust flows are working without inspecting your projects.

## What RinaWarp Does Not Send

RinaWarp does not send:

- prompts
- source code
- repository contents
- terminal output
- shell history
- file contents
- secrets, tokens, or credentials

## Opt Out

You can turn off anonymous operational telemetry in:

Settings -> Privacy & Telemetry

Telemetry failures are ignored by the app. Offline telemetry must never block runtime execution, approvals, verification, or updates.
