# RinaWarp Terminal Pro v1.8.2 Beta Tester Kit

Thank you for testing RinaWarp Terminal Pro.

## Current Status

RinaWarp Terminal Pro v1.8.2 beta now has Linux, macOS, and Windows beta artifacts. Linux is production-candidate validated. macOS and Windows builds are unsigned beta previews and may require OS security bypass steps. Production builds will be signed and notarized before full public release.

## What to Test

1. Install/open the app.
2. Select a real project workspace.
3. Ask: `Build this project and tell me what fails.`
4. Confirm Rina shows the run.
5. Confirm proof appears.
6. Export proof.
7. Quit and reopen.
8. Confirm history/proof persists.
9. Try: `Plan a fix safely.`
10. Tell us what felt confusing.

## Platform Notes

Linux should install and launch normally from the AppImage or deb package.

macOS builds are unsigned beta previews. If macOS blocks the app, right-click the app, choose Open, then choose Open again.

Windows builds are unsigned beta previews. If SmartScreen appears, choose More info, then Run anyway.

## Feedback

Use `TESTER_CHECKLIST.md` for structured feedback and `REPORTING_ISSUES.md` for crashes, broken runs, or confusing behavior.
