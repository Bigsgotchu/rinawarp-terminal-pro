# Tester Invite

Subject: RinaWarp Terminal Pro v1.8.2 beta test

Hi,

You're invited to test RinaWarp Terminal Pro v1.8.2 beta.

RinaWarp Terminal Pro is a conversational AI workbench for real developer work. It remembers useful project context and user preferences, safely plans and executes actions through controlled tools, verifies the result, and attaches proof to every meaningful run.

Current status:
RinaWarp Terminal Pro v1.8.2 beta now has Linux, macOS, and Windows beta artifacts. Linux is production-candidate validated. macOS and Windows builds are unsigned beta previews and may require OS security bypass steps. Production builds will be signed and notarized before full public release.

Please test:

3. Ask: `Build this project and tell me what fails.`
4. Export proof.
5. Quit/reopen and confirm history persists.
6. Try: `Plan a fix safely.`
7. Send feedback using the checklist.
