# Reporting Issues

Please include:

- OS and version
- App version
- Artifact used
- What you clicked/typed
- What you expected
- What happened
- Screenshot if possible
- Exported proof if available
- Diagnostic bundle if available

Do not include:

- secrets
- tokens
- private keys
- confidential file contents
- raw private source code
- private terminal output unless you have reviewed it first

## Diagnostic Bundle

If the app crashes or a run breaks, include the diagnostic/export bundle when available. If a diagnostic bundle is not available, include the exported proof and a screenshot of the visible error.
