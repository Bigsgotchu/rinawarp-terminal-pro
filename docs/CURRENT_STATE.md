# RinaWarp Terminal Pro Current State

## Product Definition

RinaWarp Terminal Pro is a natural-language AI copilot for real computer work.

It safely observes, plans, executes, verifies, remembers, and attaches Proof to meaningful actions across a user's development environment.

It is not a workbench, chatbot, terminal wrapper, code generator, CLI assistant, dashboard, or button launcher.

## Product Loop

ask → observe → plan → execute → verify → Proof → remember

Anything that does not support this loop is prototype residue and must be removed, hidden, or refactored.

## UI Truth
