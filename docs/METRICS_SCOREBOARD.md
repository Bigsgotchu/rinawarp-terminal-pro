# Metrics Scoreboard

Date: 2026-03-29

This is the minimum operating scoreboard for RinaWarp Terminal Pro and the surrounding RinaWarp Technologies revenue surface.

If a metric is not available yet, mark it as unknown explicitly rather than pretending it exists.

## Funnel Metrics

### Acquisition

- homepage visits
- pricing page visits
- download page visits
- source breakdown

### Conversion

- download clicks
- checkout start
- checkout completion
- trial start
- paid conversion rate

### Activation

- install success
- first app launch
- workspace selected
- first prompt sent
- first trusted run started
- first trusted run completed

### Retention

- day 1 active
- day 7 active
- day 30 active
- repeat trusted runs per active user
- paid retention

## Revenue Metrics

- MRR
- new paid users
- upgrade count
- downgrade count
- cancellation count
- refund count
- ARPU

## Support Metrics

- support conversations opened
- install issues
- billing issues
- entitlement issues
- average first response time
- unresolved severity-1 issues

## Product Trust Metrics

- failed trusted runs
- interrupted runs
- recovery success rate
- restore-purchase success rate
- activation failures after successful checkout

## Weekly Review Format

Every weekly review should answer:

1. what grew
2. what broke
3. where users dropped off
4. what change we will make next

## Dashboard Rule

Do not maintain more dashboards than you can actually review.

One simple source of truth reviewed every week is better than five half-maintained dashboards nobody trusts.
