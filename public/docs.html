ü§ñ AI Assistant Features

RinaWarp Terminal includes a powerful AI assistant that can help you with command suggestions, error explanations, and automation tasks.

Activating the AI Assistant

To activate the AI assistant, simply type:

rina --ai "your question here"

AI Commands

  • rina --explain - Explains the last command output
  • rina --suggest - Suggests commands based on your current directory
  • rina --fix - Helps fix common terminal errors
  • rina --automate - Creates automation scripts

üîä Voice Control

Control your terminal with voice commands for hands-free operation.

Voice Commands

  • "Navigate to [directory]" - Change directory
  • "List files" - Execute ls command
  • "Open file [filename]" - Open file in editor
  • "Run [command]" - Execute any command
  • "Clear screen" - Clear terminal output

Enabling Voice Control

Press Ctrl+Shift+V to toggle voice control on/off.
